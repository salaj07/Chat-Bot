# Chat-Bot
Real-time AI chat application using Node.js, Socket.IO, and Google Gemini API with conversational context support.
A real-time AI chatbot built using Node.js and Socket.IO, powered by Google Gemini API for intelligent, contextual conversations.
The application supports conversational memory, enabling natural back-and-forth interactions with low latency.

Designed with a simple, scalable backend architecture, this project demonstrates real-time communication, AI integration, and clean service-based code organization.
